# Отчет по лабораторной работе 4.2

## Тема

Разработка оконного приложения на Python `tkinter` для работы с циклическим односвязным списком.

## Основа работы

За основу взята лабораторная работа 4.1 из файла `100proc_last_laba4.1.c`.

## Что реализовано

В проекте сделаны три варианта работы со структурой данных:

1. Python модуль.
2. C++ модуль на динамической памяти.
3. C++ модуль на STL.

## Описание программы

Приложение позволяет:

- создать структуру;
- удалить структуру;
- добавить элемент в начало;
- добавить элемент в конец;
- вывести элементы;
- проверить пустоту;
- узнать количество элементов;
- выполнить поиск по двум полям;
- выполнить поиск по первому полю;
- выполнить поиск по второму полю;
- удалить элемент.

Все операции выполняются через отдельные модули.

## Интерфейс

Интерфейс сделан на `tkinter` и содержит:

- выбор модуля;
- поля ввода числа и строки;
- кнопки операций;
- список элементов;
- простую визуализацию на `Canvas`;
- поле сообщений.

После каждой операции интерфейс обновляет данные на экране.

## Обработка ошибок

Если пользователь пытается выполнить некорректную операцию, программа выводит понятное сообщение и не завершается аварийно.
