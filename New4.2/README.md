# New4.2

Это простое решение лабораторной 4.2 на основе файла `100proc_last_laba4.1.c`.

## Структура

- `C` - C++ модули.
- `Python` - Python модуль и два отдельных GUI приложения.

## Что сделано

- отдельное решение на Python;
- отдельное решение на C++;
- C++ модуль на динамической памяти;
- C++ модуль на STL;
- GUI на `tkinter`;
- отчет в файле `report.md`.

## Как запустить Python решение

Перейдите в папку `New4.2\Python` и выполните:

```bat
py app_python.py
```

или запустите `run_python_solution.bat`.

## Как запустить C++ решение

Перейдите в папку `New4.2\Python` и выполните:

```bat
py app_cpp.py
```

или запустите `run_cpp_solution.bat`.

При запуске `app_cpp.py` C++ модули соберутся автоматически.
